# Architecture Diagrams

This folder contains high-level architecture diagrams for the Financial Advisor Bot system.

## System Architecture
High-level component overview showing the layered architecture (UI, Service, Model, Data).

## Rendering
These diagrams use [Mermaid](https://mermaid.js.org/) syntax. GitHub renders them natively. To view them locally, you can use a Mermaid preview plugin for VS Code.
